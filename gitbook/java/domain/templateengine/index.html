模版引擎

模版引擎应该是DSL中使用最广泛的了，正所谓生成代码的代码

常见模版引擎

  • 后端: jsp, thmyleaf, groovy, freemarker
  • 前端: angularjs, vue, react

前后端分离的关键

前后端分离并不是网上说的使用json通信就是分离了，而关键是谁负责前端的渲染

模版引擎例子

  • HTML渲染: 生成静态界面，比如网页邮件
  • 代码生成技术: 生成动态SQL，比如Mybatis。生成样板代码，比如通过数据库生成DTO，Controller等

此处写的简单，但是基本上项目效率优化，从技术上减少重复劳动就基本上靠这里了
