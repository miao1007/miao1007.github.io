PlantUML for Hexo plugin released
2019-10-19 / modified at 2022-04-04 / 57 words / 1 mins
️This article has been over 3 years since the last update.

Since a lot of plugins for PlantUML diagrams are no longer maintained, I just released a new fork with compatibility for latest Hexo.

Features

  • Generate raw/base64/urlencoded svg at compile time, no external css and js required.
  • Privacy guarantee. Support rendering locally or self-hosted server.
  • Zero npm dependencies.
