Explain

调试是掌握任何框架的第一步，在Elastic中通过Explain调试为什么某个字段的分数那么高

测试数据，我们采用Demo数据中的SKU作为参数，它是keyword类型

GET kibana_sample_data_ecommerce/_search?explain=true
{
  "_source": ["sku"], 
  "query": {
    "term": {
      "sku": "ZO0450004500"
    }
  }
}
