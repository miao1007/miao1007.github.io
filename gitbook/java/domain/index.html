元编程

在满足通用技能基础后，后期业务领域的积累更加重要

业务积累

  • 对外业务: CRM, 电商, 金融, ERP等行业软件，以及互联网项目
  • 对内业务: IT建设，CIO，制造业(NPI, FMEA, SupplyChain), 电子通信的IPD流程

技术积累

  • 使用DSL面向领域编程
  • 使用BPM流程鼠标点点点编程

我个人认为随着中间件标准化，这个与业务强相关的是日后热门

  • 要有将复杂业务语法化的想法
  • 至少熟悉 Groovy, Ruby, Kotlin 中的一门JVM语言, 知道methodInvoke/missing流程是什么回事
  • Java上要会用ASM生成代码
